# Stock-Tracker-Web-App
A responsive web application that allows users to search and track real-time stock data using an external financial API. The app fetches live market data, displays price trends, and provides a clean interface for monitoring stocks.

More details:
It's a web-based application that allows users to search and track real-time stock prices using an external API.

Features:
- Search stocks by ticker
- Real-time data from Alpha Vantage API
- Clean and responsive UI
- Error handling for invalid input

Tech Stack:
- HTML, CSS, JavaScript
- REST API (Alpha Vantage)

Setup:
1. Clone the repo
2. Get a free API key from https://www.alphavantage.co
3. Replace `YOUR_API_KEY` in script.js
4. Open index.html in browser

Some future improvements Im working for:
- Add charts (Chart.js)
